============
Installation
============


Stable release
--------------

To install Kleinberg's Grid Ultimate, run this command in your terminal:

.. code-block:: console

    $ pip install kleinberg-grid-simulator

This is the preferred method to install Kleinberg's Grid Ultimate, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Kleinberg's Grid Ultimate can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/balouf/kleinberg-grid-simulator

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/balouf/kleinberg-grid-simulator/tarball/main

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .


.. _Github repo: https://github.com/balouf/kleinberg-grid-simulator
.. _tarball: https://github.com/balouf/kleinberg-grid-simulator/tarball/main
